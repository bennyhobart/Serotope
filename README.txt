src -> this is where our code goes.

	audio->music files and sounds
docs-> this is where all design documents and other information (prototypes etc..) will go

lib -> this is where all library files will go slick.jar etc... (including the java docs if the URL aren't supplied)
	jar-> contains the libraries we will use
	native-> contains the native libraries for lwjgl.jar
